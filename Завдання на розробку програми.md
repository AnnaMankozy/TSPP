# Завдання на створення програми:

Розробити програму, яка читає із файлу БД MySQL інформацію про телефони абонентів, про які відомо:

| Дані | Назва поля у БД | Формат поля у БД |
|------|----------------|------------------|
| ID | id | int(11) (Auto increment) |
| прізвище абонента | surname | varchar(50) |
| рік встановлення телефону | install_year | int(4) |
| номер телефону | phone_number | varchar(15) |
| адреса | address | varchar(100) |

Користувач – може вводити нові записи, редагувати існуючі записи та робити обробку даних за заданими параметрами, переглядати в головній формі програми перелік усіх абонентів та виконувати пошук інформації:

- за номером телефону, що вводиться у формі, виводиться прізвище та адреса абонента;
- за прізвищем та адресою виводиться номер телефону.

Значення вводяться користувачем у відповідні поля головної форми.

Результати відбору інформації користувач може зберегти у файл MS Word.

## Обмеження проекту

- Тривалість проекту із розроблення ПП – з 25.03.2026 р. до 15.06.2026 р.
- Програма має бути написана мовою C# у IDE Visual Studio із використанням фреймворку UI Windows Presentation Foundation (WPF).
- Проектування має спиратися на моделі процесів компанії у нотації IDEF + BPMN та моделі програмного продукту відповідно до RUP у нотації UML.
- Реалізація програми буде вестись методом Scrum. Product Backlog буде поділений на 2 спринта тривалістю по 2 тижні.
- По завершенні проекту програма повинна виконувати всі передбачені функції та мати документацію, перелік якої наведений у розділі «Склад програмної документації».
