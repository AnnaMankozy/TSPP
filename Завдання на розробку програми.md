# Завдання на створення програми

## Опис
Розробити програму для роботи з базою даних авіарейсів.

## Дані

| Дані | Поле | Тип |
|------|------|-----|
| ID | id | int |
| номер рейсу | number | varchar |
| місто | city | varchar |
| час | departure_time | time |
| місця | free_seats | int |

## Функції
- додавання
- редагування
- пошук
